Running the file.

1. Go to the directory the files are in.
2. Make sure the Debate_Trans.txt file and the debate_textgen.py are in the same folder.
3. Run debate_textgen.py.

Optional
-If you want to not train model to take forever change the epoch values to your liking.
-If you you want to try for a more random generation adjust temperature.
-If you want to change the amount generated edit num_generate.
